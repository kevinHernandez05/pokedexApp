# POKEDEX for Android

<p>Aplicacion nativa para android creada en Java intentando emular una Pokedex en un dispositivo Móvil.</p>

## Apis y complementos usados

- [x] Glide para las imagenes
- [x] Retrofit 
- [x] Android Studio
- [x] Un poco de café Santo Domingo
